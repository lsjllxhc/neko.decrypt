# 使用说明

## 编译

- 使用idea编译为最好的选择
- 使用JDK17及以上的版本
- 不要乱动代码部分

## 带GUI的版本

- 先装java
- 双击运行即可

## 不带GUI的版本

- 先装java（废话
- 自己输入' java -jar programname.jar --help '查看指令
