# 使用说明

## 编译

- 推荐使用idea编辑器编译
- 使用JDK17及以上版本
- 不要乱动代码

## 带GUI的版本

- 安装java
- 双击运行即可

## 不带GUI的版本

- 安装java
- 输入' java -jar programname.jar --help '查看指令
